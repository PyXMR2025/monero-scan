# Transition asio::deadline_timer to asio::steady_timer
author: vtnerd
created_at: 2026-02-14T00:41:00Z
status: merged

As of Boost 1.77, `asio::deadline_timer` has been deprecated. The build log gets spammed with warnings about it. The deprecating timer will likely be around for a while, so this isn't a high priority PR right now.

The transition to `asio::steady_timer` pushes more usage of `std::chrono` to help with type-checking. Usages of `boost::shared_ptr` are updated to `std::shared_ptr` too, when the line is already being touched to update the timer type.
