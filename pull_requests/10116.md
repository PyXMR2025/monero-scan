# dns: Add built-in DNS-over-TLS support
author: techmetx11
created_at: 2025-09-27T20:33:05Z
status: open

This PR adds built-in DNS-over-TLS (or DoT, sometimes referred to as `dot` in the source code), enabled by setting `DNS_TLS=1` in the environment variables.
It also adds a new scheme for the `DNS_PUBLIC` parser: `tls` that accepts the IP of the DNS server and the expected domain of the DNS server, used to validate the given SSL certificate. (e.g. `tls://9.9.9.9#dns.quad9.net`)
