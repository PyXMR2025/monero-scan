<h2>10.10.2</h2>
<ul>
<li><strong>FIX</strong>: BetterEm: Add better support for <code>*em, **em,strong***</code> and <code>_em, __em,strong___</code> cases.</li>
<li><strong>FIX</strong>: Caret: Add better support for <code>*sup, **sup,ins***</code>.</li>

</ul>
<h2>10.10.1</h2>
<ul>
<li><strong>FIX</strong>: FancyLists: Remove a mistaken semicolon from injected classes.</li>
</ul>
<h2>10.10</h2>
<ul>
<li><strong>NEW</strong>: FancyLists: Add new FancyLists extension.</li>
<li><strong>NEW</strong>: MagicLink: Change social links to support <code>x</code> instead of <code>twitter</code>. <code>twitter</code> is still recognized but is
now deprecated and will be removed at a future time.</li>
<li><strong>NEW</strong>: Emoji: Update Twemoji data to the latest.</li>
<li><strong>FIX</strong>: PathConverter: Fixes for latest changes in Python regarding <code>urlunparse</code>.</li>
</ul>
