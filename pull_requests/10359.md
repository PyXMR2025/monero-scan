# [DO NOT MERGE] FCMP++: Rust FFI + crypto conversion `selene_scalar_from_bytes`
author: j-berman
created_at: 2026-03-11T22:21:12Z
status: open

Builds on top of:
- #10358

This PR introduces the Rust FFI, as well as a function to take 32 bytes -- the byte representation of a Selene scalar -- and convert the byte representation into the Selene scalar object used in the Rust code.

Recall "Selene" is the name of the curve in the tower cycle towering Ed25519: https://gist.github.com/tevador/4524c2092178df08996487d4e272b096 . As part of constructing the curve tree merkle tree, we take Ed25519 points and convert them to Selene scalars (the leaves of the tree are composed of Selene scalars).

The `selene_scalar_from_bytes` function introduced in this PR takes the 32 byte representation (that we derive from ed25519 points on the C++ side), and produces the Selene scalar Rust FFI compatible object that we use in future Rust FFI functions when constructing the curve tree merkle tree.
