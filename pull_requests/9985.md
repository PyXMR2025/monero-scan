# crypto: check+throw for Cryptonight v1 invalid input
author: jeffro256
created_at: 2025-07-11T16:36:07Z

If `crypto::cn_slow_hash()` is called with `variant=1` and an input length of less thab 43 bytes, it triggers a program exit. This checks first and throws an exception instead.
