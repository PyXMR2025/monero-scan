# Meta: Define Criteria For Wallets (PC & Mobile)
author: jermanuts
created_at: 2024-11-07T18:17:02Z
status: open

For the page https://www.getmonero.org/downloads/

Minimum Requirements:
doesn't require any PII (Personally Identifiable Information)
non-custodial wallet
must receive regular updates.
client must be open source
No tracking / can opt-out
uses standard tx generation

Best Case:
software received a security audit
provides hashes and binaries are signed
reproducible builds
SOCKS5 proxy support
Can lock wallet with passcode & encrypts database

(this list is from matrix discussions)

This comparison chart by Feather wallet dev may give us some more ideas https://monerosecurity.org/
