# **OPTIONAL** plain text transaction
author: asbai
created_at: 2025-10-17T15:55:21Z
status: open

We all know that when transferring XMR (or any other cryptocurrency) to a CEX, an anonymous transfer is of little significance, as the exchange will certainly know who you are and how much money you have received. Therefore, could an **optional** feature be added to allow users to use plaintext transactions when transferring funds to KYC-compliant custodial wallets, such as those used by CEXs (**Off by default**).

Meanwhile, since transfers between non-custodial XMR wallets would still maintain the default anonymity, this approach could potentially allow more CEX listings without compromising XMR's privacy protection.
