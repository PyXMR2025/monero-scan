# [DO NOT MERGE] FCMP++: point to cycle scalar impl
author: j-berman
created_at: 2026-03-11T23:59:40Z
status: open

Builds on top of:
- #10361

Introduces the functions to convert a Helios point to Selene scalar (`helios_point_to_selene_scalar`), and Selene point to Helios scalar (`selene_point_to_helios_scalar`), which are critical when growing the curve trees merkle tree.

The curve trees merkle tree's leaf layer is composed of Selene scalars. To get the next layer's elements, we hash a chunk of leaves (Selene scalars) at a time, and that hash is a point on the Selene curve. Then to get the next layer's elements, we convert each of those Selene points into Helios scalars using `selene_point_to_helios_scalar`, and hash a chunk of those at a time to get Helios points. Then to get the next layer's elements, we convert each of those Helios points into Selene scalars using `helios_point_to_selene_scalar`, and hash a chunk of those at a time to get Selene points. And so on, alternating between Selene and Helios at each layer.
