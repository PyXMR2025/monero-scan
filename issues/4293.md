# 18.3.2 GUI Shows Network Status Disconnected When Connecting to Own Node Running Locally (So I Downgraded GUI to 18.3.1) - UPDATE: Will be resolved in 18.3.3
author: ajsingh007
created_at: 2024-03-17T15:43:00Z

Hope this helps other users...

But I upgraded my remote node running locally as well as my GUI to 18.3.2 (I'm on MacOS, Intel). 

On the 18.3.2 GUI the Network Status kept saying "Disconnected" whether my node was running 18.3.2 or 18.3.1

When I downgraded the GUI back to 18.3.1, it connects just fine whether my node is running 18.3.2 or 18.3.1

I'm not sure why this is but I hope it can come in handy to others who have the same issue!
