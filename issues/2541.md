# Add Cake Wallet and remove MyMonero
author: sethforprivacy
created_at: 2025-10-10T19:12:32Z
status: closed

Per https://old.reddit.com/r/Monero/comments/1nzq8p1/announcement_mymonero_is_sunsetting_stewardship/, MyMonero is sunsetting operations.

Understandably [some Monero community members](http://old.reddit.com/r/Monero/comments/1nzq8p1/announcement_mymonero_is_sunsetting_stewardship/nisxhaz/?context=3) have checked getmonero.org to verify and noticed that Cake Wallet itself is not listed, and MyMonero still is. This PR simply rectifies that as well as updating platform support for Monero.com as we have had Windows support for some time.
