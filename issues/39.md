# Network simulation library
author: b-g-goodell
created_at: 2018-10-02T03:33:20Z
status: open

Started [here](https://github.com/b-g-goodell/research-lab/tree/master/source-code/Poisson-Graphs), a simulation of a coin-agnostic cryptocurrency network can be used as a sandbox/playground for testing new approaches to consensus or difficulty. These simulations should be fleshed out more, better commented, made visually appealing, but more importantly: useful information needs to be extractable from the transcripts of the simulations.
