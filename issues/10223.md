# [POC] StageX release builds
author: tobtoht
created_at: 2025-11-22T22:34:26Z
status: open

This is just an experiment for now. I'm still evaluating whether [StageX](https://stagex.tools/) can (eventually) replace Guix for release builds. I will expand on the pros and cons as this PR reaches maturity.

This currently uses scaffolding from Guix builds, which can be cleaned up.

GCC: 13.1.0
Clang: 20.1.8

| Target | Builds | 
|--------|--------|
| x86_64-linux-musl | ✓ |
| aarch64-linux-musl | ✖ |
| riscv64-linux-musl | ✖ |
| x86_64-w64-mingw32 | ✖ |
| x86_64-unknown-freebsd | ✓ |
| x86_64-apple-darwin | ✓ |
| arm64-apple-darwin | ✓ | 
| aarch64-linux-android | ✓ |
