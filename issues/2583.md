# Add Skylight Wallet to Light Wallets section
author: SlowBearDigger
created_at: 2026-01-04T17:57:02Z
status: closed

This pull request adds Skylight Wallet to the Monero downloads page. Skylight Wallet is a non-custodial, privacy-focused light wallet developed by Magic Grants.

Changes include:
- Addition of the Skylight Wallet launcher icon (skylight.png).
- Entry in the "Local synchronization" table with platform indicators (Android, iOS, Linux, Windows, macOS).
- Inclusion in the mobile wallets list for consistency.
- Links directed to the official GitHub repository.
