# add-exclusive-node with only 1 node will only queue 1 span at a time
author: nahuhh
created_at: 2025-12-30T02:49:59Z
status: open

As the title says.

if you have more than 1 add-exclusive-node set, it will continue downloading to fill the queue, even if all of the downloading is from 1 node.

if you only have 1 exclusive peer, it will downloads 1 span, and waits to sync it before downloading the next one.
