# verify-merge.py : detailed file path / folder name checking.
author: jonathancross
created_at: 2019-12-09T22:10:21Z
status: closed

This takes care of 2 of the 3 checks listed in #40 -- namely:
1. Verification of all path components.
2. Verification that each `.sig` has a `.assert` file and vice versa.

### Example error messages:

User name error:
```
Verifying file path naming...
ERROR: User 'foo' doesn't have a matching PGP key.  Expected gitian-pubkeys/foo.asc
 * Found in path: v0.15.0.1-win/foo/monero-win-0.15-build.assert
```
Signature without an `.assert` file:
```
Verifying file path naming...
ERROR: One or more sig files doesn't have a matching assert file:
  - v0.15.0.1-win/foo/monero-win-0.16-build.assert.sig
  - v0.15.0.1-win/foo/monero-win-0.15-build.assert.sig
```
Assert file without a corresponding `.sig` file:
```
Verifying file path naming...
ERROR: Assert file found without corresponding sig file:
v0.15.0.1-win/foo/monero-win-0.15-build.assert
```
Version number (or platform, or other general path error):
```
Verifying file path naming...
ERROR: File path appears to be incorrect:
v0.15.0.1-win/stefanomarty/monero-win-0.16-build.assert
Expected:
v0.15.0.1-win/stefanomarty/monero-win-0.15-build.assert
ERROR: One or more file paths failed verification.
```
