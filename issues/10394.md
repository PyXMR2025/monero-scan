# brew: remove unnecessary homebrew/bundle tap
author: ratchips
created_at: 2026-03-31T02:57:27Z
status: closed

`homebrew/bundle` tap is deprecated (now empty) and causes `brew bundle` to fail:

```bash
$ brew update && brew bundle --file=contrib/brew/Brewfile
...
Tapping homebrew/bundle
Error: homebrew/bundle was deprecated. This tap is now empty and all its contents were either deleted or migrated.
Tapping homebrew/bundle has failed!
...
`brew bundle` failed! 1 Brewfile dependency failed to install
```

`brew bundle` is built into Homebrew, so the tap line is no longer needed. Removed.
