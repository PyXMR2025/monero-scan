# Missing IPv6
author: 86dd
created_at: 2024-08-11T22:45:38Z

Hello,
getmonero.org is lacking IPv6 connectivity.
getmonero.org has IPv6 (cloudflare), but is simply a redirect to
www.getmonero.org, which has no IPv6.
lists.getmonero.org and downloads.getmonero.org also do not have IPv6.
It would be nice to have this added, to allow IPv6 only clients and nodes to access the website.
