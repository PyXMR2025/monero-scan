# Revokable branched outputs for uniform timelocked contracts
author: ghost
created_at: 2020-06-07T15:48:09Z
status: open

Revokable branched outputs would allow for atomic swaps, inheritance outputs and other more complex escrow transactions all while keeping Monero transactions uniform and using already existing cryptographic primitives. I have made a whitepaper detailing the proposal/idea [here](https://github.com/philogy/revokable-branched-outputs). Would love to hear your feedback and thoughts!
