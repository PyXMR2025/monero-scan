As posted in https://forum.getmonero.org/6/ideas/2575/implement-icon-or-user-doc-re-rss-category-thread-notification:

I would find it both helpful and an overall improvement to forum usability if either/or both of the following were implemented:
- an RSS icon for each category and/or thread
- easily-visible documentation on how to append `/feed` to a URL (as noted in [https://forum.getmonero.org/11/meta/85/notifications-on-threads](https://forum.getmonero.org/11/meta/85/notifications-on-threads))

Presently, the only RSS icon available will feed solely from [https://getmonero.org/feed.xml](https://getmonero.org/feed.xml).
