## 🎨 Theme System Enhancement

This PR introduces a comprehensive theme system for the Monero website with three beautiful theme options that prioritize user privacy and experience.

### ✨ Features Added

**🌈 Three Theme Options:**
- **Light Theme**: Clean white backgrounds with dark text (default)
- **Dark Theme**: Dark gray backgrounds with light text for reduced eye strain
- **Cyberpunk Theme**: Futuristic black/purple backgrounds with neon green text and cyan accents

**🔒 Privacy-First Implementation:**
- **No localStorage or cookies** - themes persist via URL parameters only
- **Transparent system** - theme choice visible in URL (`?theme=dark` or `?theme=cyberpunk`)
- **User controllable** - can be disabled by removing URL parameter
- **Bookmarkable** - theme preferences can be saved and shared

**📱 Complete Integration:**
- **Main navigation integration** - Theme selector as standard menu item
- **Desktop and mobile support** with synchronized controls
- **Automatic link updating** to maintain theme across navigation
- **Responsive design** for all screen sizes

### 🛠️ Technical Implementation

**Modern Web Standards:**
- **CSS Variables system** for maintainable theming
- **Hybrid CSS + JavaScript approach** - CSS handles visual switching, JS manages persistence
- **Pure CSS fallback** - themes work even if JavaScript is disabled
- **Cross-browser compatible** - works on all modern browsers

**Performance Optimized:**
- **Minimal JavaScript footprint** - only for URL management
- **CSS-only visual switching** using `:checked` and `:has()` selectors
- **No runtime performance impact** on theme switching
