# Add Haveno DEX to P2P exchanges page
author: arnoldnakamura
created_at: 2026-03-16T06:28:54Z
status: open

## Summary

- Add Haveno (https://haveno.exchange) to the P2P exchanges section on the merchants page
- Haveno is a decentralized P2P exchange forked from Bisq, redesigned for Monero-native trading
- Supports fiat payment methods (cash, bank transfers) with 2-of-3 multisig escrow
- AGPL-3.0 licensed, non-custodial, no KYC, Tor support
- Active development with multiple community instances (RetosSwap, DawnSwap, others)
- GitHub: https://github.com/haveno-dex/haveno

## Changes

- `community/merchants/index.md`: Added Haveno card matching existing format (4th card, right half)
- `_i18n/en.yml`: Added `havenodescr` and `visithaveno` translation keys
- `img/merchants/haveno.png`: Added Haveno logo (landscape, from official repo)

## Notes

Haveno is widely considered the most actively developed P2P DEX for Monero and is notably absent from this page despite being listed elsewhere in the Monero ecosystem. It supports direct XMR-to-fiat trades via cash-by-mail and face-to-face payment methods — something neither Bisq nor BasicSwapDEX currently offer. Multiple production instances are live with significant trade volume.
